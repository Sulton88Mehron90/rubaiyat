Dive into the world of poetic wisdom with the **Rubaiyat of Omar Khayyam**. Inspired by the profound verses of Omar Khayyam, this application provides a refreshing glimpse into the timeless wisdom encapsulated in his quatrains.

### <span style="color:#2E8B57;">Description</span>

**Rubaiyat** offers a captivating experience for both poetry enthusiasts and wisdom seekers. Users are presented with random quatrains, merging the age-old insights of Omar Khayyam with the digital age.

### <span style="color:#2E8B57;">Features</span>

- **Random Verse Generation**: Each visit offers a fresh perspective through random quatrain generation.
- **Elegant Design**: A minimalistic and elegant design ensures readers can focus entirely on the verse.

### <span style="color:#2E8B57;">Setup & Installation</span>

To get **Rubaiyat of Omar Khayyam** running on your machine:

1. Clone the repository: `git clone git@github.com:Sulton88Mehron90/rubaiyat.git`
2. Enter the project directory: `cd rubaiyat`
3. Install dependencies: `npm install`
4. Launch the app: `npm start`
